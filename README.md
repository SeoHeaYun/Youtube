# Yuktube
유튜브 API와 디자인을 기반으로 한 앱

## 프로젝트 소개
* 안드로이드의 각 기능을 익히고 깔끔한 UI를 보여주기 위해 제작하였습니다.
* 여러 카테고리와 가장 인기있는 영상을 볼 수 있으며 검색과 좋아요를 할 수 있습니다.

## 개발 기간
